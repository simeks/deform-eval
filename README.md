# Dependencies

* deform
* ANTs
* SimpleITK

# Setup

* `export DEFORM_EVAL_DATA_PATH <path>` (default: 'data')
* `export ANTS_BIN_PATH <path>`
* `export DEFORM_BIN_PATH <path>` (default: 'bin')
* `python run.py setup <data/Hammers_mith-n30r95/>`

# Run

* `python run.py ants`
* `python run.py df_gpu`
* `python run.py df_cpu`
